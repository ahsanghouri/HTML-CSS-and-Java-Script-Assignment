PIAIC

Presidential initiative for artificial intelligence and cloud computing


Presidential initiative and Artificial intelligence and Computing (PIAIC) is a privately funded not-for-profit educational initiative started by President of Pakistan , Dr. Arif Alvi, Kazi Rahat Ali, and Zia Khan to help promote education, research and business opportunites in emerging tech fields like

  • Cloud Native Computing
  • Artificial Intelligence
  • Block Chain
  • Internet of Things

cloud native computing:

The cloud native computing program prepares the students to program in open source technlogies like, linux, docker, kubernetes, java script, node.js, react and react native while introducing the students to development of Microservices and software development techniques on cloud platforms like AWS
